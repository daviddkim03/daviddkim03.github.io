## Overview

Grading and reporting on student assessments was a repetitive, manual process. The Report Generator replaces it with an automated pipeline: assessment results flow in from ClassMarker, are processed and organized through the Google Sheets API, and come out as finished reports — no manual grading pass required.

## Technical Highlights

- **ClassMarker API integration**: pulls assessment results automatically.
- **Google Sheets API integration**: structured storage and processing of grade data.
- **Data scraping**: fills the gaps where APIs don't expose what the reports need.
- **Report automation**: generates finished academic reports without manual assembly.

## Outcome

Eliminated the repetitive manual grading workflow end to end, freeing staff time and removing transcription errors from the reporting process.

## Technologies Used

- **Python**: data processing and API integration
- **JavaScript / Node.js**: automation pipeline and report generation
